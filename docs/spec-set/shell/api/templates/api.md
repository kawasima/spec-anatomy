### エラーレスポンス

| HTTPステータス | エラーコード             | 業務的意味                              | 関連する Core のエラー型                             |
| -------------- | ------------------------ | --------------------------------------- | ---------------------------------------------------- |
| 400            | INVALID_INPUT            | リクエストの形式不正                    | (Shell 側の zod / Bean Validation 等)                |
| 401            | UNAUTHORIZED             | 認証エラー                              | (Shell 側の認証ミドルウェア)                         |
| 403            | INSUFFICIENT_ROLE        | 権限エラー                              | Core の権限エラー型 ([../../../spec-model/...]())    |
| 404            | RESOURCE_NOT_FOUND       | リソース不在                            | (Shell 側の Repository から)                         |
| 409            | INVARIANT_VIOLATION      | 状態不整合エラー                        | Core の状態不整合エラー型                            |
| 409            | <Core のエラーコード>    | <Core のエラーの業務的意味>             | Core のエラー型                                      |
| 503            | SERVICE_UNAVAILABLE      | 外部システム応答なし                    | (Shell 側のリトライ / サーキットブレーカー)          |

}
