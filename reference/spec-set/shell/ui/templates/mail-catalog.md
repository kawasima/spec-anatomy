## 命名規約

メールIDは次の形式で付けます。

- `MAIL-{連番3桁}` または `MAIL-{業務カテゴリ}-{連番3桁}`
- 例: `MAIL-101`、`MAIL-ORDER-001`

ファイル名は `MAIL-{ID}-{kebab-case-name}.md` の形式にします。
