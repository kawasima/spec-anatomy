## 命名規約

帳票IDは次の形式で付けます。

- `REPORT-{連番3桁}` または `REPORT-{業務カテゴリ}-{連番3桁}`
- 例: `REPORT-101`、`REPORT-ACCOUNTING-001`

ファイル名は `REPORT-{ID}-{kebab-case-name}.md` の形式にします。
