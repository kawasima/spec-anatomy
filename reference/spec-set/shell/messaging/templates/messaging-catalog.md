## 命名規約

メッセージングIDは次の形式で付けます。

- `MSG-{連番3桁}` または `MSG-{業務カテゴリ}-{連番3桁}`

トピック名/キュー名は次の形式が一般的です。

- ドメインイベント: `<集約名>.<イベント名>`（例: `order.approved`）
- コマンド: `<集約名>.<コマンド名>.requested`

ファイル名は `MSG-{ID}-{kebab-case-name}.md` の形式にします。
